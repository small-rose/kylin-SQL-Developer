# Kylin PL/SQL Developer 快捷键速查表

## 文件操作

| 快捷键 | 操作 |
|--------|------|
| `Ctrl+N` | 新建 SQL 文件 |
| `Ctrl+O` | 打开 SQL 文件 |
| `Ctrl+S` | 保存 |
| `Ctrl+Shift+S` | 另存为 |
| `Ctrl+W` | 关闭当前标签 |

## 编辑操作

| 快捷键 | 操作 |
|--------|------|
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` | 重做 |
| `Ctrl+F` | 查找 |
| `Ctrl+H` | 替换 |

## SQL 操作

| 快捷键 | 操作 |
|--------|------|
| `F8` | 执行当前 SQL 语句（光标所在分号段，自动检测；选区优先） |
| `Ctrl+Shift+F10` | 执行当前 SQL 语句（与 F8 等效） |
| `Ctrl+E` | 执行计划 |
| `Ctrl+Shift+F` | 格式化 SQL |

## 导航

| 操作 | 说明 |
|------|------|
| 左侧对象浏览器 | 双击对象 → 数据预览；右键 → DDL/DML/展开包 |
| 点击 `DATABASE` 标签 | 折叠/展开左侧对象树 |
| 右侧 FILES 面板 | 列出已保存 SQL 文件，双击打开 |
| 右侧 THUMBNAIL 面板 | 缩略图，点击/拖拽跳转到编辑器对应行 |
| 多标签页 | 点击标签切换；× 关闭 |

## SQL 编辑器

| 特性 | 说明 |
|------|------|
| 语句自动检测 | 光标定位到 SQL 语句上自动高亮当前块（分号分隔），按 F8 执行 |
| 执行结果标记 | 执行后在左侧行号区显示绿色 ✓ 或红色 ❗ 图标 |
| 执行日志 | 底部"消息"面板记录每次执行的时间/耗时/行数 |

## 数据库连接

| 操作 | 说明 |
|------|------|
| 管理连接按钮 | 新建/编辑/测试/删除数据库连接 |
| 每个标签独立连接 | 标签页顶部工具栏选择连接和 Schema |
| 查询超时 | 连接属性可设置 SQL 执行超时时间（0=不限） |
